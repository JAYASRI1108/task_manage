About TaskFlow

TaskFlow is a frontend-based task management system built using HTML, CSS, JavaScript, and Bootstrap. It helps users organize work efficiently without the need for backend or database integration.

The system stores task data using browser localStorage and provides reports, reminders, and priority-based task analysis.
